Login to Signal K

The first step is to configure Signal K server authentication (a pre-existing Signal K server username and password is required). Go to KIP's Connectivity options, toggle Login to server, enter your Signal K credentials and click Sign in. Once completed, click the upper right X button to get back to this dashboard (Connectivity options).

  • If you don’t have a Signal K account, ask your captain or access the Signal K Admin Settings / Users page and create a user.
  • If you see an error, double-check your username and password, and make sure your device is on the same network as your Signal K server.

Changing Layouts

Follow these simple steps to learn how to change your dashboard layout:
  1. Resize this Tutorial widget using the now visible dashed borders or corners arrows. Resize it so it does not take up all the screen. You need free space in your dashboard to add more widgets. You can delete this Tutorial widget later when you are done with the instructions.
  2. Touch & Hold or Click & Hold in an empty area to open the Add Widget selection panel.
  3. Choose a widget from the selection panel to add it to the dashboard.

Possible widget operations

  • Touch and drag to move widgets
  • Drag corners and sides to resize widgets
  • Double Tap on a widget to configure it
  • Press and Hold on a widget to delete or duplicate widgets

Exiting dashboard edit mode

Use the Save or Cancel buttons in the lower right corner to either persist your changes or revert changes.

Adding more dashboards

Swipe left and hit the Dashboards button. From this page you can add, rename, order, delete, and duplicate entire dashboards.
