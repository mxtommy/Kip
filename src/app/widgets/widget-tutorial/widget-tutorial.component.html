Welcome to KIP! Here are some basic gestures to help you navigate the application:

Actions Gestures Keyboard Shortcuts

Open Actions menu Swipe left Right Arrow
Open Notification menu Swipe right Left Arrow

Toggle Night mode N/A Atl + Ctrl + n
Toggle Dashboard edit mode N/A Atl + Ctrl + e

Create Your Own Layout

First, configure Signal K server authentication (a pre-existing Signal K server user/password is required). Swipe left to open the Actions menu, toggle Login to server switch enter you pre-existing Signal K server credentials.

Dashboards (individual layout pages) are "Locked" by default. This is to prevent accidental modifications. Edit the dashboard by swiping left to open the Actions menu and hit the bottom unlock button. You can later add and manage dashboards using the Dashboards option from the Actions menu.

Build Your Layout

First, resize this Tutorial widget using the now visible dashed borders so it does not take up all the screen. Then, to add a widget, in an empty dashboard area, press and hold. This will open the widget selection menu. Here you can choose from a variety of widgets.

To manage widgets, you can:

  • Touch and drag to move widgets
  • Drag corners and sides to resize widgets
  • Double Tap on a widget to configure it
  • Press and Hold on a widget to delete or duplicate widgets
Exit Dashboard Edit Mode

Use the Save or Cancel buttons in the lower right corner to either persist your changes or revert back all changes.
