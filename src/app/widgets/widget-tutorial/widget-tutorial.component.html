Build Your Layout

First, resize this Tutorial widget using the now visible dashed borders so it does not take up all the screen. You can delete this Tutorial widget later when you are done with the instructions. Then, to add a widget: in an empty dashboard area, press and hold. This will open the widget selection panel. Here you can choose from a variety of widgets.

To manage widgets, you can:

  • Touch and drag to move widgets
  • Drag corners and sides to resize widgets
  • Double Tap on a widget to configure it
  • Press and Hold on a widget to delete or duplicate widgets

Exit Dashboard Edit Mode

Use the Save or Cancel buttons in the lower right corner to either persist your changes or revert changes.

Add More Dashboards

Swipe left and hit the Dashboards button. From this page you can add, rename, position and duplicate dashboards.
