Getting Started

Welcome to KIP! Here are some basic gestures to help you navigate the application:

Actions Touch Mouse Keyboard Shortcuts
Open Actions menu Swipe left Click, drag left, and release Shift + Ctrl + ← (Left Arrow)
Open Notification menu Swipe right Click, drag right, and release Shift + Ctrl + → (Right Arrow)

Toggle Fullscreen N/A N/A Shift + Ctrl + F

Note that the words Touch and Tap are synonymous with mouse click.

Log in to Signal K

The first step is to configure Signal K server authentication (a pre-existing Signal K server username and password is required). Use the table above to find the gesture, mouse or keyboard shortcut to open the Actions menu, then select Settings. Toggle Login to server and enter your Signal K credentials.

  • Why log in? Logging in allows KIP to do two things: 1) to send commands to Signal K and 2) to save its configuration on the server so it can be shared between multiple devices. It is the recommended method to use KIP.
  • If you don’t have a Signal K account, ask your Signal K administrator, access the Signal K Admin Settings / Users page and create a user or check your server’s documentation for setup instructions.
  • After entering your credentials, select Login or Connect. If successful, your connection status will update and live data will appear in KIP.
  • If you see an error, double-check your username and password, and make sure your device is on the same network as your Signal K server.

Changing Layouts

Follow these simple steps to learn how to change your dashboard layout:
  1. Resize this Tutorial widget using the now visible dashed borders or corners arrows. Resize it so it does not take up all the screen. You need free space in your dashboard to add more widgets. You can delete this Tutorial widget later when you are done with the instructions.
  2. Touch & Hold or Click & Hold in an empty area to open the Add Widget selection panel.
  3. Choose a widget from the selection panel to add it to the dashboard.

Possible widget operations

  • Touch and drag to move widgets
  • Drag corners and sides to resize widgets
  • Double Tap on a widget to configure it
  • Press and Hold on a widget to delete or duplicate widgets

Exiting dashboard edit mode

Use the Save or Cancel buttons in the lower right corner to either persist your changes or revert changes.

Adding more dashboards

Swipe left and hit the Dashboards button. From this page you can add, rename, order, delete, and duplicate entire dashboards.
