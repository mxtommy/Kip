All New Touch, Mouse, and Keyboard Navigation

Heads up: The new version introduces a completely redesigned fullscreen interface with no navigation bar. Please review the input modes table below to get an understanding of the new ways to operate KIP before upgrading. After the upgrade, give yourself time to get comfortable with the changes.

Actions Touch Mouse Keyboard Shortcuts
Open Actions menu Swipe left Click, drag left, and release Shift + Ctrl + ← (Left Arrow)
Open Notification menu Swipe right Click, drag right, and release Shift + Ctrl + → (Right Arrow)
Cycle through dashboards Swipe up/down Click, drag up/down, and release Shift + Ctrl + ↑/↓ (Up/Down Arrow)
Toggle Fullscreen N/A N/A Shift + Ctrl + F
Toggle Night mode N/A N/A Shift + Ctrl + N
Toggle dashboard edit mode N/A N/A Shift + Ctrl + E
Note that the words Touch and Tap are synonymous with mouse click.

Migration Limitations

Important: The new widget layout engine is completely different from the previous version. When you upgrade, your configuration will be migrated, but the way dashboards and widgets are arranged will change significantly.

  • All KIP configuration settings will be migrated.
  • Each v2 Page layout will become a v3 Dashboard.
  • All widgets and their settings will appear in their new dashboards.
  • Note: Widget positions, order, and sizes WILL BE LOST. After upgrading, dashboards will contain small widgets that you must reposition and resize manually.

To upgrade your configuration, follow these steps:

  1. As a safety measure, if you are using Signal K authentication, manually back up your configuration. Make a copy of the 9.0.0.json files found on the Signal K server in the following folders:
    • ~/.signalk/applicationData/users/(username)/kip/
    • ~/.signalk/applicationData/global/kip/
  2. Click the Upgrade button below.
  3. Once the upgrade is complete, KIP will restart and display the migrated dashboards.
  4. Reposition and resize your widgets as needed.
  5. Once satisfied, delete the 9.0.0.json configuration files located in the previously mentioned folders. They are not needed any more.
