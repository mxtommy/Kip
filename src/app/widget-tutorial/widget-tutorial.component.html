Kip Installation Notes

For complete functionality, certain Signal K plugin should be installed on the server you are connecting to:

  • @signalk-alarm-silencer: To silence Notifications on the server. This enables Kip to instruct the Signal K server to silence said alarm for on all devices supporting Signal K Notifications, else Kip will be silenced.
  • @signalk/signalk-autopilot: required for the N2k Autopilot to function.
    Use the Kip N2k autopilot at your own risk. Use with caution and constant supervision.

Initial Configuration

Two options are available:

  1. Load Demo Configuration

    If you want to dive right in to see what KIP can do, you can use this button to load demo configuration with simulation data from demo.signalk.org. Note that an internet connection is required to receive simulation data:



  2. Editing Page Layout

    Page layouts are "Locked" by default. This is to prevent accidental touch screens interaction from changing anything. A page layout is a screen where you can add, configure and remove the various avalable widgets such as gauges and components. Multiple pages can be added to facilitate organization. You can unlock page layout using the hamburger icon in the lower right corner and selecting "Configuration -> Edit Layout". Go Ahead try it now!

Next Steps...

Configuring Page Layout

Now that the page layout is unlocked, you can do a few actions:

  • The icon in the lower right can be used to select the Widget you would like to display
  • The icon in the lower left can be used to "split" the pages into various areas. You can use area to display different widgets on one page. Areas can be resized to better fit designs
  • The "X" in the upper right can be used to delete an area (basically undoing a split)
  • Once a widget is selected, there will be a gear in the lower middle. Clicking it will allow you to setup the widget parameters
